CSS Selectors

CSS selectors are used to select the HTML elements you want to style.They help in targeting HTML elements and applying styles effectively Here are some common types of CSS selectors:

  • Simple
  • Combinator
  • Attribute
  • Pseudo Classes
  • Pseudo Elements

Simple Selectors

Simple selectors have again different types ,they are:

Element Selector
element selector applies styles to all occurrences of a specific HTML tag Example:p,h1,h2,etc
Class Selector
Selects all elements with a specific class attribute. Example:(.)
ID Selector
Selects a single element with a specific ID attribute. Example:(#)
Group Selector
Selects multiple elements by separating each selector with a comma. Example:(p,.ab,#xy)
Universal Selector
applies styles to all elements Example:(*)
